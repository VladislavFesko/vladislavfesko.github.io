Welcome to Gammiux

Gammiux is a custom Linux operating system which is based on Debian. It comes with many apps and Openbox window manager. Also, it has the taskbar LXPanel and dock Plank. This operating system are absolutely free, so you can download it for free.

Features

  • Many preinstalled apps
  • Lightweight desktop based on Openbox
  • Based on the latest Debian version
