Welcome to VladOS!

VladOS is a operating system based on Kubuntu. This OS have many applications, themes and wallpapers.

VladOS builds with Systemback - a special tool for create live ISO on Ubuntu (formerly VladOS builds with Bodhibuilder - a Remastersys fork).

VladOS is works only on amd64 computers.
